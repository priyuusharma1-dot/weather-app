A simple and responsive Weather App that provides real-time weather updates for any location. This application fetches live weather data using an API and displays key information such as temperature, humidity, wind speed, and weather conditions in a user-friendly interface.

🔹 Features:

* 🌍 Search weather by city name
* 🌡️ Displays temperature, humidity, and wind speed
* ☁️ Shows current weather conditions with icons
* ⚡ Fast and responsive design
* 📱 Mobile-friendly UI

🔹 Technologies Used:

* HTML
* CSS
* JavaScript
* Weather API (e.g., OpenWeatherMap)

🔹 How to Use:

1. Enter the city name in the search bar
2. Click on the search button
3. View the current weather details instantly

This project is perfect for beginners learning API integration and front-end development.
